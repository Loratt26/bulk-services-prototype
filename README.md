# Bulk Services — Cowlendar Prototype

Live: https://bulk-services.briansalazar.dev
Case study: https://briansalazar.dev/en/case-studies/bulk-services-cowlendar

Functional 5-step wizard prototype for bulk service creation in Cowlendar — Penida.io's booking app for Shopify. Designed and built by Brian Salazar during feature validation with the CTO.

## Stack

- Next.js 15 · React 18 · TypeScript · Tailwind · Shopify Polaris

## Local dev

```bash
npm install
npm run dev
```
